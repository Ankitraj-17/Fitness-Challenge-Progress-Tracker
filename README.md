# Fitness Challenge Progress Tracker

## Setup
```bash
pip install matplotlib
```

## Run
```bash
cd fitness_tracker
python fitness_tracker_main.py
```

## Files
- `fitness_tracker_main.py` - Main application
- `participant_class.py` - Participant OOP class
- `fitness_utils.py` - Utility functions with decorator, lambda, list comprehensions
- `activity_logs.csv` - Activity data storage
- `participant_summary.json` - Summary data (auto-generated)

## Features
- ✅ Add participants
- ✅ Log daily activities (steps, exercise, calories)
- ✅ View statistics
- ✅ Visualize progress with charts
- ✅ Data validation with decorator
- ✅ CSV and JSON storage
